m-BNISmarter

Project information

  • Category: Mobile Apps
  • Client: BNI Corporate University
  • Project URL: PlayStore & AppStore

m-BNISmarter adalah aplikasi berbasis mobile yang berguna untuk mendapatkan informasi terkait pelatihan atau pembelajaran untuk pegawai BNI. Serta dilengkapi oleh beragam fitur seperti Deep46, Absensi, WFH Learning, More Unleash, Event, Learning Game, Video, dan Notifikasi

Aplikasi ini mempunyai beberapa fitur antara lain:

- Akses Materi dan Ujian Deep46 Berbasis Mobile
Pegawai dapat melihat materi dan mengikuti ujian Deep46 melalui aplikasi. Kemudian mendapatkan hasil di menu profile masing-masing

- QR Code Presensi, WFH Learning, Learning Game dan lainnya
Sistem Presensi Pelatihan untuk peserta dan fasilitator yang telah terintegrasi dengan QR Code, pembelajaran secara WFH, Learning Game Quiz dan lainnya
